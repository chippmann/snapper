A snapping FlingBehavior for LazyListState. Typically this would be created via rememberSnapperFlingBehavior.

Note: the default parameter value for decayAnimationSpec is different to the value used in rememberSnapperFlingBehavior, due to not being able to access composable functions.

Parameters

lazyListState

The LazyListState to update.

decayAnimationSpec

The decay animation spec to use for decayed flings.

springAnimationSpec

The animation spec to use when snapping.

snapOffsetForItem

Block which returns which offset the given item should 'snap' to. See SnapOffsets for provided values.

maximumFlingDistance

Block which returns the maximum fling distance in pixels. The returned value should be >= 0.

endContentPadding

The amount of content padding on the end edge of the lazy list in pixels (end/bottom depending on the scrolling direction).
